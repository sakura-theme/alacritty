## Installation

Clone this repository.

```
git clone https://github.com/sakura-theme/alacritty.git ~/path/to/alacritty
```

Add following lines to `~/.config/alacritty/alacritty.yml`.

```
import:
  - ~/path/to/alacritty/sakura.yml
```
